# GreenNest — React JSX Version

Standard Vite + React project. Every component and page uses `.jsx`; every component keeps its own CSS file.

## Start

1. `npm install`
2. Copy `.env.example` to `.env.local` and add Firebase values.
3. Enable Email/Password and Google authentication in Firebase.
4. `npm run dev`

Plant data: `src/data/plants.json`.
